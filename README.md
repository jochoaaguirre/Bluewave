# BlueWave Swim

## Descripción del Proyecto
BlueWave Swim es una tienda en línea especializada en artículos para nadadores. El proyecto incluye la implementación de una página de inicio atractiva y un catálogo de productos con imágenes, descripciones y precios, utilizando React.
